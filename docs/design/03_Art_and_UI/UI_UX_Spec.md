## 一、设计原则

### 1.1 核心设计原则

**双模式适配性**：同一界面框架需同时支持休闲探索模式（湿画法/明亮温暖）与生存挑战模式（干画法/冷峻严肃）。

**写实水彩风格统一**：
-   所有UI底板必须使用**水彩纸纹理 (Cold Press Paper)**。
-   边框和分割线必须呈现**手绘笔触**，避免绝对直线。
-   图标和插图需保留**铅笔草稿痕迹**和**边缘柔和**的特征。

**无障碍优先**：所有交互元素需满足WCAG 2.1 AA标准，支持色盲、视障用户。

**性能优化**：界面加载时间≤2秒，动画帧率≥60FPS，内存占用≤200MB。

### 1.2 交互设计规范 (PC键鼠版)

**响应时间标准**：
- 点击反馈：≤100ms
- 页面切换：≤300ms
- 数据加载：≤2s
- 动画过渡：200-400ms (使用Ease-in-out，模拟墨水扩散)

**键鼠交互规范**：
- **点击热区**：≥32x32px (鼠标精确操作)，重要按钮 ≥48x48px
- **悬停反馈 (Hover)**：所有可交互元素必须有明显的悬停状态变化（颜色晕染扩散、边缘加深）。
- **右键操作**：支持右键点击查看详情/上下文菜单。
- **滚轮操作**：支持列表滚动、地图缩放。
- **快捷键**：核心功能需支持键盘快捷键（如 I-背包, M-地图, Esc-返回）。

## 二、统一编号体系与组件命名

### 2.1 界面编号规则

采用**T-[模式]-[功能模块]-[序号]**格式：
- T-CA-LOGIN-001：休闲模式登录界面
- T-SU-MAIN-001：生存模式主界面
- T-SH-SET-001：共享设置界面

### 2.2 组件命名规范

**基础组件**：
```
BTN_[功能]_[状态]      // 按钮组件 (手绘边缘)
TXT_[内容]_[类型]      // 文本组件 (衬线体/手写体)
IMG_[内容]_[尺寸]      // 图片组件 (水彩插画)
CARD_[类型]_[状态]     // 卡片组件 (纸质纹理)
BAR_[类型]_[方向]      // 进度条组件 (颜料流动效果)
```

**状态后缀**：
```
_DEFAULT    // 默认状态
_HOVER      // 悬停状态 (水彩晕染)
_ACTIVE     // 激活状态 (颜色加深)
_DISABLED   // 禁用状态 (褪色/干枯)
_LOADING    // 加载状态
_ERROR      // 错误状态 (红色墨渍)
```

### 6.3 进度条规范

**线性进度条**（Linear Progress）：
```css
.progress-linear {
  height: 8px;
  background: #E0E0E0; /* 纸张底色 */
  border-radius: 4px;
  overflow: hidden;
}

.progress-fill {
  height: 100%;
  background: linear-gradient(90deg, #4CAF50 0%, #66BB6A 100%); /* 水彩渐变 */
  border-radius: 4px;
  transition: width 0.3s ease;
}
```

**圆形进度条**（Circular Progress）：
```css
.progress-circular {
  width: 48px;
  height: 48px;
  border: 4px solid #E0E0E0;
  border-top: 4px solid #4CAF50;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}
```

### 6.4 卡牌UI规范 (Card UI)

卡牌是游戏的核心交互载体，需体现独特的“实体纸牌感”与“水彩艺术性”。

**基础尺寸**（Card Dimensions）：
- **标准卡牌**：180px × 260px (宽高比 9:13)
- **卡牌间距**：水平间距 16px，选中时放大 1.1 倍

**卡牌结构**（Card Structure）：
1.  **底板 (Base)**：所有卡牌必须使用 `Art_Visual_Guide` 定义的 `.material-paper` 水彩纸纹理。
2.  **边框 (Frame)**：根据稀有度区分边框颜色与笔触复杂度。
3.  **插图 (Illustration)**：位于卡牌上部，占高度的 55%，边缘需做羽化或水彩遮罩处理。
4.  **信息区 (Info)**：包含标题（手写体）、标签、数值图标。

**稀有度样式规范**（Rarity Styles）：

| 稀有度 | 边框颜色变量 | 视觉特征 | 阴影强度 |
| :--- | :--- | :--- | :--- |
| **Common** (普通) | `--tt-gray-500` (深灰) | 单线墨迹，绳索/帆布纹理，极简风格 | Level 1 |
| **Rare** (稀有) | `--ca-secondary` (青/蓝绿) | 双层晕染，蕨类植物与溪流波纹，角落水渍 | Level 2 |
| **Epic** (史诗) | `--ca-culture-3` (靛/蓝紫) | 山峰剪影，星空渐变，气旋与岩层纹理 | Level 3 |
| **Legendary** (传说) | `--ca-sun` (镉黄/金) | 太阳光芒，金箔等高线，古指南针纹饰，动态流光 | Level 4 |

**图标框规范**（Icon Frames）：
- **形状**：非标准矩形，四角带有随机的“毛边”或“溢出”效果。
- **背景**：半透明白色 (`rgba(255,255,255,0.8)`) 叠加纸纹。
- **描边**：使用 `2px` 宽度的手绘线条，透明度 `80%`。

**模式差异化**：
- **休闲模式**：卡牌边缘整洁，色彩明亮，阴影柔和 (`blur 8px`)。
- **生存模式**：卡牌边缘有磨损、烧焦或水渍痕迹，阴影锐利 (`blur 2px`)。
