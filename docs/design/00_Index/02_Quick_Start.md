## 🎯 5分钟快速了解项目

### 项目定位

《徒踪-V2.0》是一款**双模式休闲+硬核生存游戏**：
- 🎴 **休闲探索模式**：大湾区文化+卡牌消除
- ⚠️ **生存挑战模式**：全国风险路线+生存管理

### 核心创新点

1. **双模式架构**：两种完全不同的游戏体验
2. **数据互通**：共享货币、装备、成就
3. **教育价值**：文化探索+户外安全教育
4. **真实性**：基于真实地理和户外案例

### 技术栈

- **引擎**：Godot 4.6
- **语言**：GDScript（主）+ C# 12.0（性能关键）
- **平台**：Steam PC（首发）→ 移动端（未来）

## 🔑 核心概念速查

### 双模式架构

```
《徒踪-V2.0》
    │
    ├─ 休闲探索模式（Casual Mode）
    │   ├─ 大湾区10条路线
    │   ├─ 5属性管理（体能、饥饿、口渴、体温、心率）
    │   ├─ 照片收集+文化探索
    │   └─ 容错率70%，轻松愉快
    │
    ├─ 生存挑战模式（Survival Mode）
    │   ├─ 全国10条风险路线
    │   ├─ 6属性管理（+心智）
    │   ├─ 极端天气+危机事件
    │   └─ 容错率30%，硬核挑战
    │
    └─ 共享层
        ├─ 三种货币（环保值、爬升值、徒步数）
        ├─ 照片卡收集
        ├─ 装备系统
        └─ 成就系统
```

---

### 三货币体系

| 货币 | 定位 | 获取难度 | 主要用途 |
| --- | --- | --- | --- |
| **环保值** | 日常流通 | 容易 | 消耗品、基础装备 |
| **爬升值** | 长期养成 | 困难 | 稀有装备、永久提升 |
| **徒步数** | 里程积累 | 中等 | 装备升级、路线解锁 |

---

### 核心系统清单

**必须理解的10个核心系统**：

1. ✅ **双模式架构系统** - 模式切换和数据同步
2. ✅ **点击消除引擎** - 核心玩法机制
3. ✅ **卡牌系统** - 卡牌数据和视觉
4. ✅ **路线节点系统** - 路线和节点管理
5. ✅ **资源与经济系统** - 三货币体系
6. ✅ **属性系统** - 5属性（休闲）/6属性（挑战）
7. ✅ **天气系统** - 天气影响和演变
8. ✅ **事件系统** - 随机事件触发
9. ✅ **存档系统** - 数据持久化
10. ✅ **成就系统** - 长期目标

## ⚡ 常见陷阱与注意事项

### ⚠️ 陷阱1：混淆两种模式的设计

**问题**：将休闲模式的设计应用到挑战模式，或反之

**解决**：
- 两种模式完全独立，不要混淆
- 休闲模式：轻松、无压力、文化探索
- 挑战模式：硬核、高压力、生存挑战

---

### ⚠️ 陷阱2：忽略术语统一

**问题**：使用"海拔数"而不是"爬升值"

**解决**：
- 统一使用"爬升值"（Climbing Points）
- 查阅术语表（GDD附录A）
- 代码中使用`climbing_points`

---

### ⚠️ 陷阱3：破坏经济平衡

**问题**：随意修改货币获取或消耗数值

**解决**：
- 任何数值修改前先查阅经济系统设计文档
- 使用配置文件而非硬编码
- 修改后进行平衡性测试

---

### ⚠️ 陷阱4：忽略性能优化

**问题**：大量实例化对象导致性能下降

**解决**：
- 使用对象池技术（第9章9.3.1节）
- 动态加载/卸载资源
- 使用Godot Profiler监控性能

## 🔧 开发环境配置

### 必需工具

1. **Godot 4.6**
   - 下载：https://godotengine.org/
   - 版本：4.6稳定版

2. **代码编辑器**（可选）
   - VS Code + Godot插件
   - 或使用Godot内置编辑器

3. **版本控制**
   - Git
   - 推荐使用Git LFS管理大文件

### 项目配置

**project.godot关键配置**：

```ini
[application]
config/name="徒踪-V2.0"
run/main_scene="res://scenes/main/main_menu.tscn"
config/features=PackedStringArray("4.6", "Forward Plus")

[autoload]
GameManager="*res://scripts/core/GameManager.gd"
ModeManager="*res://scripts/core/ModeManager.gd"
DataManager="*res://scripts/core/DataManager.gd"
# ... 其他自动加载

[display]
window/size/viewport_width=1920
window/size/viewport_height=1080
window/size/mode=2

[rendering]
renderer/rendering_method="forward_plus"
textures/canvas_textures/default_texture_filter=2
```
