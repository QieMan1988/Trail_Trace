# PC版全流程界面布局蓝图 (Scheme A 优化版)

**文档版本**：v1.1.0
**基准分辨率**：1920 x 1080 (16:9)
**设计风格**：方案 A (仪表盘式) 进化版
**适用平台**：PC (键鼠操作为主)

---

## 一、 核心设计理念：自适应仪表盘 (Adaptive Cockpit)

基于团队选择的 **方案 A**，我们针对 PC 2D 环境进行了深度优化。核心思路是保持“信息直观可见”的优点，但根据模式不同，切换仪表盘的**视觉皮肤**和**信息密度**。

*   **生存模式** = **战术终端 (Tactical Terminal)**
    *   风格：高对比度、科技感/硬核户外风格、暗色背景。
    *   信息：六维属性全显，心率波形动态跳动，详细的环境参数。
*   **休闲模式** = **旅行手账 (Travel Journal)**
    *   风格：拟物化（纸张/相机）、暖色调、明亮背景。
    *   信息：属性简化为图标，侧重显示连击数、收集进度和风景。

## 二、 核心玩法界面布局 (Core Gameplay)

### 2.1 布局网格 (Grid System)
屏幕被划分为 **12列 x 12行** 的网格。
*   **顶栏 (Top Bar)**: Row 0-1 (高度 ~180px)
*   **底栏 (Bottom Bar)**: Row 10-11 (高度 ~180px)
*   **主视窗 (Main View)**: Row 2-9 (高度 ~720px)
*   **侧边栏 (Side Panels)**: Col 0-2 (左) & Col 10-11 (右) [可选展开]

**优化点**：
1.  **左侧监控面板**：将 Scheme A 的底部仪表盘移至**左侧中段**，符合 PC 玩家视线习惯（类似 RPG 的队伍状态栏），且不遮挡脚下路况。
2.  **心率波形图**：在左侧面板顶部独立显示，作为生存模式的核心视觉压力源。
