## 一、设计原则

### 1.1 核心设计原则

**双模式适配性**：同一界面框架需同时支持休闲探索模式（明亮温暖）与生存挑战模式（冷峻严肃）

**无障碍优先**：所有交互元素需满足WCAG 2.1 AA标准，支持色盲、视障用户

**性能优化**：界面加载时间≤2秒，动画帧率≥60FPS，内存占用≤200MB

### 1.2 交互设计规范 (PC键鼠版)

**响应时间标准**：
- 点击反馈：≤100ms
- 页面切换：≤300ms
- 数据加载：≤2s
- 动画过渡：200-400ms

**键鼠交互规范**：
- **点击热区**：≥32x32px (鼠标精确操作)，重要按钮 ≥48x48px
- **悬停反馈 (Hover)**：所有可交互元素必须有明显的悬停状态变化（高亮、放大、光标变化）
- **右键操作**：支持右键点击查看详情/上下文菜单
- **滚轮操作**：支持列表滚动、地图缩放
- **快捷键**：核心功能需支持键盘快捷键（如 I-背包, M-地图, Esc-返回）

## 二、统一编号体系与组件命名

### 2.1 界面编号规则

采用**T-[模式]-[功能模块]-[序号]**格式：
- T-CA-LOGIN-001：休闲模式登录界面
- T-SU-MAIN-001：生存模式主界面
- T-SH-SET-001：共享设置界面

### 2.2 组件命名规范

**基础组件**：
```
BTN_[功能]_[状态]      // 按钮组件
TXT_[内容]_[类型]      // 文本组件
IMG_[内容]_[尺寸]      // 图片组件
CARD_[类型]_[状态]     // 卡片组件
BAR_[类型]_[方向]      // 进度条组件
```

**状态后缀**：
```
_DEFAULT    // 默认状态
_HOVER      // 悬停状态
_ACTIVE     // 激活状态
_DISABLED   // 禁用状态
_LOADING    // 加载状态
_ERROR      // 错误状态
```

## 四、交互状态与反馈

### 4.1 通用交互状态

**按钮状态**：
```css
/* 默认状态 */
.btn-default {
  opacity: 1.0;
  transform: scale(1.0);
  transition: all 0.2s ease;
}

/* 悬停状态 */
.btn-hover {
  opacity: 0.8;
  transform: scale(1.05);
}

/* 激活状态 */
.btn-active {
  opacity: 0.6;
  transform: scale(0.95);
}

/* 禁用状态 */
.btn-disabled {
  opacity: 0.4;
  cursor: not-allowed;
}
```

**加载状态**：
- 骨架屏：灰色占位块
- 旋转动画：60RPM
- 进度条：带百分比显示

## 五、适配方案

### 5.1 分辨率适配

**基础断点**：
```css
/* 桌面端标准 */
@media (min-width: 1920px) {
  .container { max-width: 1440px; margin: 0 auto; }
}

/* 笔记本适配 */
@media (max-width: 1366px) {
  .scale-factor { transform: scale(0.71); }
}
```

### 5.2 平台适配 (PC)

**Steam平台**：
- 支持Steam Overlay
- 集成Steam成就系统
- 支持云存档同步

**控制器支持**：
- Xbox手柄：A键确认，B键返回
- PlayStation手柄：×键确认，○键返回
- 键盘：方向键导航，Enter确认，Esc返回

## 六、无障碍设计

### 6.1 视觉无障碍

**色彩对比度**：
- 普通文本：≥4.5:1
- 大文本：≥3:1
- 交互元素：≥4.5:1

**色盲支持**：
- 提供色盲友好配色方案
- 使用纹理+颜色双重编码
- 支持自定义配色

### 6.2 听觉无障碍

**音效替代**：
- 提供视觉反馈替代音效
- 关键信息文字提示

### 6.3 操作无障碍

**键盘导航**：
- 支持Tab键顺序导航
- 提供快捷键设置
- 支持自定义按键绑定

**辅助功能**：
- 屏幕阅读器支持
- 高对比度模式
- 字体大小调节（12px-24px）

## 七、性能标准

### 7.1 加载性能

**界面加载时间**：
- 首屏加载：≤2秒
- 界面切换：≤300ms
- 图片加载：≤1秒
