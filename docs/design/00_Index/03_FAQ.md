## <a name="基础问题"></a>一、游戏基础问题

### Q1：《徒踪-V2.0》是什么类型的游戏？

**A**：《徒踪-V2.0》是一款创新的**双模式游戏**，融合了：
- **休闲探索模式**：轻松的卡牌消除+大湾区文化探索
- **生存挑战模式**：硬核的资源管理+全国风险路线生存

两种模式共享货币和成就，但玩法完全独立。

---

### Q2：游戏的核心玩法是什么？

**A**：核心玩法是**点击消除卡牌**，但两种模式有不同侧重：

**休闲模式**：
- 点击消除卡牌，收集照片和文化知识
- 追求高连击和高分
- 轻松愉快，无压力

**挑战模式**：
- 点击消除卡牌获取资源（水、食物、装备）
- 管理6种生存属性（体能、饥饿、口渴、体温、心率、心智）
- 应对极端天气和危机事件
- 紧张刺激，策略性强

---

### Q3：游戏适合什么样的玩家？

**A**：游戏设计了双模式架构，适合不同类型的玩家：

| 玩家类型 | 推荐模式 | 理由 |
| --- | --- | --- |
| 休闲玩家 | 休闲探索模式 | 轻松解压，无失败惩罚 |
| 硬核玩家 | 生存挑战模式 | 真实挑战，策略深度 |
| 文化爱好者 | 休闲探索模式 | 大湾区文化知识丰富 |
| 户外爱好者 | 生存挑战模式 | 真实户外风险模拟 |
| 收集向玩家 | 两种模式结合 | 照片卡、装备、成就收集 |

---

### Q4：游戏的目标平台是什么？

**A**：
- **首发平台**：PC (Windows 10/11) - Steam
- **未来计划**：移动端 (iOS 12+, Android 8.0+)
- **开发引擎**：Godot 4.6

## <a name="双模式问题"></a>二、双模式相关问题

### Q5：两种模式有什么区别？

**A**：两种模式在多个维度上完全不同：

| 维度 | 休闲探索模式 | 生存挑战模式 |
| --- | --- | --- |
| **核心玩法** | 卡牌消除+照片收集 | 资源管理+生存决策 |
| **属性系统** | 5属性（体能、饥饿、口渴、体温、心率） | 6属性（+心智） |
| **路线背景** | 大湾区10条经典路线 | 全国10条高风险路线 |
| **难度定位** | 轻松休闲，容错率70% | 硬核挑战，容错率30% |
| **游玩时长** | 10-15分钟/局 | 20-30分钟/局 |
| **失败惩罚** | 无惩罚，可重试 | 死亡重置路线进度 |
| **教育主题** | 文化探索+环保教育 | 户外安全+生存技能 |

---

### Q6：两种模式的数据是否互通？

**A**：**部分互通**，具体如下：

**✅ 完全共享的数据**：
- 三种货币（环保值、爬升值、徒步数）
- 照片卡收集进度
- 装备和道具
- 成就系统
- 角色等级和经验

**❌ 独立的数据**：
- 关卡进度（各模式独立推进）
- 路线解锁（各模式独立）
- 模式特定资源（如休闲模式的环保值存量、挑战模式的当前属性）

---

### Q7：可以随时切换模式吗？

**A**：**不可以随时切换**，切换有限制：

**✅ 允许切换的时机**：
- 主界面（游戏大厅）
- 路线选择界面
- 结算界面

**❌ 禁止切换的时机**：
- 关卡进行中
- 商店界面
- 战斗/消除进行时

**原因**：保持模式纯粹性和沉浸感，避免玩家利用切换规避挑战。

---

### Q8：是否必须玩两种模式？

**A**：**不强制**，但建议体验两种模式：

- 可以只玩一种模式，不影响游戏进度
- 但会错过另一种模式的专属内容和成就
- 部分协同成就需要两种模式都有进展
- 两种模式的货币可以互相支持（如休闲模式赚的徒步数可以在挑战模式购买装备）

## <a name="货币系统"></a>三、货币与经济系统

### Q9：三种货币有什么区别？

**A**：三种货币定位不同：

**环保值（Eco Points）**：
- **定位**：日常流通货币
- **获取**：容易（每局300-800）
- **用途**：购买消耗品、基础装备、照片卡包
- **特点**：快速获取，快速消费

**爬升值（Climbing Points）**：
- **定位**：长期养成货币
- **获取**：困难（每局2-10个）
- **用途**：稀有装备、永久提升、解锁内容
- **特点**：稀缺资源，用于重要提升

**徒步数（Hiking Distance）**：
- **定位**：里程积累货币
- **获取**：中等（每局1000-2500）
- **用途**：装备升级、路线解锁
- **特点**：持续积累，衡量游玩时长

---

### Q10：三种货币可以互相兑换吗？

**A**：**不可以**。三种货币各有用途，不提供兑换功能，原因：
- 防止玩家刷货币破坏经济平衡
- 保持爬升值的稀缺性
- 鼓励玩家体验完整游戏循环

---

### Q11：货币有上限吗？

**A**：
- **环保值**：无上限，但鼓励消费
- **爬升值**：无上限，稀缺资源
- **徒步数**：无上限，里程积累

---

### Q12：如何快速获得爬升值？

**A**：爬升值获取途径：

**高效方法**：
1. **完美通关**：每关+5爬升值（所有属性>60%）
2. **首次通关**：每关+10爬升值（仅首次）
3. **成就解锁**：10-100爬升值（完成特定成就）
4. **挑战模式**：收益是休闲模式的2-5倍

**建议策略**：
- 新手先玩休闲模式熟悉玩法
- 熟练后挑战生存模式获取更多爬升值
- 追求完美通关和成就解锁

## <a name="休闲模式"></a>四、休闲探索模式

### Q13：休闲模式的5属性和挑战模式有什么不同？

**A**：休闲模式使用**心率**属性，挑战模式使用**心智**属性：

**心率（休闲模式）**：
- 反映运动强度
- 消除卡牌时心率上升
- 休息时心率下降
- 心率过高会影响体能恢复

**心智（挑战模式）**：
- 反映精神压力
- 恶劣环境和危机事件降低心智
- 心智过低会影响决策准确率
- 心智归零会导致精神崩溃死亡

---

### Q14：如何获得三星评价？

**A**：三星评价需要同时满足三个条件：

1. **完成时间** < 15分钟
2. **剩余体能** > 80
3. **连击次数** > 50次

**建议策略**：
- 保持高连击减少体能消耗
- 合理使用补给道具
- 规划最优消除路径
- 熟悉卡牌分布规律

---

### Q15：如何快速收集照片卡？

**A**：照片卡获取途径：

**主要途径**：
1. **观景点拍照事件**：固定节点，100%获得
2. **关卡奖励**：每关1张
3. **三星通关**：额外1张随机照片卡
4. **商店购买**：50-300环保值/张

**高效策略**：
- 优先消除照片卡（15%概率获得）
- 追求三星通关
- 完成照片相关成就
- 定期购买照片卡包

---

### Q16：连击系统的最佳策略是什么？

**A**：连击系统技巧：

**基础技巧**：
- 在3秒内连续消除可维持连击
- 连锁消除自动计入连击

**高级策略**：
1. **规划消除顺序**：先消除容易的，留难的最后
2. **利用连锁**：消除会触发连锁的卡牌
3. **保持节奏**：不要停顿超过3秒
4. **特殊消除**：7连击后触发全屏特效，额外获得照片卡

**连击收益**：
- 5-9连击：体能消耗-5%，资源获取+10%
- 10-19连击：体能消耗-10%，资源获取+25%
- 20+连击：体能消耗-15%，资源获取+50%

## <a name="生存模式"></a>五、生存挑战模式

### Q17：生存模式的6属性如何管理？

**A**：6属性管理是生存模式的核心：

**优先级排序**：
1. **体能**：最重要，归零立即死亡
2. **口渴**：次重要，消耗最快
3. **体温**：受天气影响大，需及时调整
4. **饥饿**：消耗较慢，但影响体能
5. **心率**：受运动和天气影响
6. **心智**：长期影响，需定期恢复

**管理策略**：
- 时刻关注体能，低于50立即休息
- 优先补充水源（口渴消耗最快）
- 根据天气携带对应装备
- 定期休息恢复心智

---

### Q18：如何应对极端天气？

**A**：不同天气有不同应对策略：

**暴风雪**：
- 必备装备：防寒服（体温+2°C）
- 策略：寻找避难所，减少移动
- 风险：失温概率50%，迷路概率40%

**热浪**：
- 必备装备：防暑服（体温-1°C）
- 策略：多补充水分，避免中午行动
- 风险：中暑概率40%，脱水加速

**暴风雨**：
- 必备装备：雨衣（体温+1°C，防滑50%）
- 策略：减速前进，避免滑倒
- 风险：失温概率50%，滑倒概率60%

**沙尘暴**：
- 必备装备：防尘面具+护目镜
- 策略：寻找避风处，用湿布捂住口鼻
- 风险：迷路概率50%，呼吸道感染30%

---

### Q19：死亡后会失去什么？

**A**：死亡后的损失：

**✅ 保留**：
- 所有货币（环保值、爬升值、徒步数）
- 所有收集（照片卡、装备图鉴）
- 所有成就
- 角色等级和经验

**❌ 失去**：
- 当前路线进度（重置到起点）
- 当前关卡的资源和属性
- 本局的通关奖励

**特殊情况**：
- 如果触发了救援结局，可以保留50%进度
- 如果触发了牺牲结局，虽然死亡但获得额外奖励

---

### Q20：如何提高生存率？

**A**：提高生存率的关键策略：

**准备阶段**：
1. 携带足够的水和食物（至少3瓶水+5份食物）
2. 根据路线携带对应装备（防寒/防暑/防水）
3. 必备装备：指南针、急救包

**游玩阶段**：
1. 时刻关注属性，任一属性<50立即处理
2. 遇到极端天气优先寻找避难所
3. 不要贪图进度，生命比速度重要
4. 合理使用休息和补给

**危机应对**：
1. 迷路时原地等待，不要盲目前进
2. 失温时立即寻找避难所+使用防寒装备
3. 脱水时立即饮水，不要尝试危险方法
4. 属性接近致命阈值时考虑求援

## <a name="货币系统"></a>六、货币与经济系统

### Q21：为什么设计三种货币而不是一种？

**A**：三种货币满足不同层次的需求：

1. **环保值**：日常消费，快速循环，保持游戏节奏
2. **爬升值**：长期成长，稀缺资源，提供成就感
3. **徒步数**：里程积累，衡量投入，装备升级专用

这样设计可以：
- 给玩家更多策略选择
- 避免单一货币导致的经济失衡
- 提供多层次的成长体验

---

### Q22：爬升值会不会太难获得？

**A**：爬升值的获取速率经过精心设计：

**轻度玩家**（1小时/天）：
- 日均获得：10-15爬升值
- 一周可获得：70-105爬升值
- 足够购买：1个角色解锁+部分永久提升

**中度玩家**（2小时/天）：
- 日均获得：25-35爬升值
- 一周可获得：175-245爬升值
- 足够购买：装备槽扩展+稀有装备

**重度玩家**（4小时/天）：
- 日均获得：50-70爬升值
- 一周可获得：350-490爬升值
- 足够购买：多项高级提升+传说装备

---

### Q23：两种模式的货币获取差异会导致玩家只玩一种模式吗？

**A**：不会，原因如下：

**平衡设计**：
- 挑战模式收益更高（1.6-2.0倍），但难度也更大
- 休闲模式更轻松，适合快速积累
- 时间效率相近（收益/时间比）

**激励机制**：
- 每日任务鼓励体验两种模式
- 协同成就需要两种模式都有进展
- 模式切换奖励（+200徒步数）

**玩家选择**：
- 根据心情和时间选择模式
- 休闲模式积累资源，挑战模式获取稀有奖励
- 两种模式互补，体验更丰富

- 所有核心管理器使用自动加载

**A**：平衡性测试标准：

**调整方法**：
